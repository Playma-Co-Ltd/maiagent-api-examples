# Python doc

## python version

3.11

## install requirements

1. create virtual env

```bash
python3.11 -m venv .venv
```

2. activate virtual env

3. (Optional) install uv
```bash
pip install uv
```

4. install requirements

if install uv
```bash
uv pip install -r requirements.txt
```

if not install uv

```bash
pip install -r requirements.txt
```
