## 拿取 API Key

請登入 [MaiAgent 系統後台](https://admin.maiagent.ai/)，根據以下步驟取得 API Key：

1. 於右上角點擊「使用者名稱」
2. 點擊「帳號」
3. 即可查看「API 金鑰」
